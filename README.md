# Introduction to HTTP

During this module you will learn:
- Basics of HTTP, TCP/IP, DNS, URL, URI
- Commands, Status Codes, Headers, Forwarding

and you will be introduced to the tools like:
- curl, 
- wget, 
- ping, 
- telnet, 
- ssh, 
- wireshark, 
- postman
